Referencia para construção do site

Os recursos utilizados no desenvolvimento do site são a a recusividade, tendo como objetivo em adptar-se a qualque dispositivel móvel, notebook ou desktop.

Utilização de recurso de destaque nos links que sinalizam ao serem pré-selecionadas.

Também possíve visualizar os recurso de CSS implementados em arquivos distinto do html. Isso possibilita um desenvolvimento web mais limpo e organizado, tendo em vista que os arquivos html fica na raiz principal, as imagens, arquivos css, arquivos java script e demais recursos em pastas distintas com objetivo de organização e localização dos dados utilizados.

Ferramentas utilizada foram o Visual code como editor de código, gimp e Krita como editores de imagens e o navegador Google Chrome para realizar os testes de desenvolvimento.

Durante o desenvolviemto foi utilizado o material de apoio apostila fornecida pelo professor, sites educaionas que auxiliam no aprendizagem de tecnologia da informação e cainais educativo diponíveis no youtube.

Linux é um termo popularmente empregado para se referir a sistemas operativos ou sistemas operacionais que utilizam o Kernel Linux. O núcleo foi desenvolvido pelo programador finlandês Linus Torvalds, inspirado no sistema Minix. Wikipédia

Família de SO: Sistema operacional tipo Unix

Data De Lançamento: 17 de setembro de 1991

Licença: GNU GPLv2 / outras (o nome "Linux" é uma marca comercial)

Gravado em: C, Linguagem assembly

Interfaces de usuário padrão: Shell do Unix, KDE Plasma 5, MATE, Cinnamon, Unity, LXDE, elementary OS, Xfce

Métodos de atualização: KernelCare, dpkg, GNOME Software
